Nepirity Analytics is a plugin that provides web traffic data analysis feature with data gathered from Google Analytics.

== Description ==
Do you want to analyze your Wordpress website?

Nepirity Analytics lets you analyze your website with features integrated with Google Analytics core reporting API.

Check how many people visited your website, from what search engine they came, what keyword they typed. Check whether visitors like your new content, what content is most popular.

Nepirity Analytis provides its analysis report on many pages.

Before you install, you can experience the plugin on the demo site below.

[https://demo.nepirity.com/wp-admin](https://demo.nepirity.com/wp-admin)

== Installation ==

1. Copy the plug-in files to the wp-content/plugins directory.
2. Activate Nepirity Analytics.
3. Follow the instructions given by the Nepirity Setup Wizard.

Please, visit the site below for more information.

https://www.nepirity.com/blog/nepirity-analytics

== Frequently Asked Questions ==
= Do I have to manually insert Google Tracking code into my website? =
No, check Tracking Option under Settings to automatically insert Google Tracking code.

= Is the plugin free? =
The plugin is free to use. You can experience more advanced features such Behavior Analysis and Realtime Analysis by purchasing license.
